# ☕ Java & POO Studies

Este repositório foi criado para documentar minha jornada de aprendizado em **Java** e **Programação Orientada a Objetos (POO)** durante as aulas práticas de Engenharia de Software.

Aqui, você encontrará os códigos desenvolvidos em sala de aula, exercícios de fixação e pequenos projetos focados na construção de uma base sólida em desenvolvimento back-end.

## 🎯 Objetivos do Repositório
- Praticar a sintaxe da linguagem Java.
- Compreender e aplicar os pilares da Programação Orientada a Objetos.
- Registrar a evolução técnica desde os conceitos mais básicos até aplicações mais complexas.

## 📚 Conceitos Abordados
Até o momento, os códigos deste repositório exploram os seguintes temas:
- Criação de Classes e Instanciação de Objetos.
- Definição de Atributos e Métodos.
- Manipulação de dados e exibição no console (`System.out.println`).
- *[Adicione mais tópicos conforme as aulas forem avançando, ex: Encapsulamento, Herança, Polimorfismo]*
